# Backend Demo Endpoints

This folder contains a lightweight PHP backend prototype for the milestone check.

## Endpoints

- `POST /backend/login.php`
- `POST /backend/logout.php`
- `GET /backend/me.php`

## Demo Accounts

- `admin@eldms.local` / `AdminPass123!`
- `exporter@eldms.local` / `ExporterPass123!`
- `logistics@eldms.local` / `LogisticsPass123!`
- `customs@eldms.local` / `CustomsPass123!`

The login endpoint accepts either form data or a JSON body with `email` and `password`.
